# 奇货可查网站还原项目 (QHKC Web Clone)

## 项目简介

这是一个完整的期货数据分析平台网站还原项目，目标是精确还原 [奇货可查](https://x.qhkch.com/) 网站的所有功能和设计。

## 技术栈

- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite
- **样式方案**: TailwindCSS 4
- **UI组件库**: shadcn/ui
- **图表库**: ECharts + echarts-for-react
- **路由**: Wouter
- **包管理器**: pnpm

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

### 构建生产版本

```bash
pnpm build
```

### 启动生产服务器

```bash
pnpm start
```

## 重要提示

1. **ECharts图表提取**: 使用特殊方法提取原网站的ECharts配置
2. **风格精确匹配**: 必须精确匹配原网站的颜色、间距、边框
3. **逐个菜单开发**: 每完成一个菜单需要确认后再继续

## 开发规范

- 使用 TypeScript 进行类型检查
- 遵循 React 最佳实践
- 使用 TailwindCSS 进行样式开发
- 组件化开发，提高代码复用性
